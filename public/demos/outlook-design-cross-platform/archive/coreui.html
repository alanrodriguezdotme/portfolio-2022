Core UI

Our brand identity & personality, the aesthetic nature of our product experiences, are compositional in that they're composed of elements arranged in relation to each other, creating a greater whole, or harmony.

Revealing Core UI

Deconstructing these compositions into individual constituent parts would reveal core UI - the foundational graphic design elements such as grid, typographic ramps, leading, kerning, color pallets, iconography, graphical styles, and core controls.

Core UI is both visible and non-visible elements used to create compositions

Microsoft accent colors

Accent colors should be used as a starting point when secondary color is needed, but should be complimentary to an app's core brand color.

Swim lanes

Core or accent colors can be used as a base to create swim lanes. These are shades of a color created by adding percentage tints of black or white.
